# Azure Cloud E-Commerce Platform | Go-retailops

This project is a high-performance, modern E-commerce platform built with a Go backend and a premium, responsive frontend.

## 🚀 Technologies
- **Backend**: Go (API) - Fast, scalable, and memory-efficient.
- **Frontend**: HTML5, Vanilla JavaScript, Tailwind CSS (CDN).
- **Aesthetics**: Modern "Clean" design with hover animations, glassmorphism, and soft shadows.
- **Infrastructure**: Azure-ready Terraform configuration for cloud deployment.

## 🛠️ How to Run

### 1. Start the Go Backend
Ensure you are in the backend directory and run:
```bash
cd backend/go-api
go run main.go
```
*The API will start on `http://localhost:8081`. Connects to items and handles checkout logic.*

### 2. Launch the Storefront
Simply open the following file in your favorite web browser:
`frontend/react-app/index.html`

## 📦 Features
- **Live Product Gallery**: Fetching high-quality product data from the Go API.
- **Interactive Shopping Bag**: Add/remove items with a real-time summary.
- **Micro-animations**: Smooth transitions, hover effects, and spring-based animations.
- **Azure Ready**: Pre-configured Terraform for cloud deployment on Microsoft Azure.
